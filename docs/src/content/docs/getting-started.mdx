Integrate Typesense search into Starlight

## Prerequisites

You will need to have a Starlight website set up.
If you don't have one yet, you can follow the ["Getting Started"](https://starlight.astro.build/getting-started) guide in the Starlight docs to create one.

## Installation

import { Steps, Tabs, TabItem } from '@astrojs/starlight/components';

<Steps>

1. `starlight-typesense` is a Starlight [plugin](https://starlight.astro.build/reference/plugins/). Install it by running the following command in your terminal:

   <Tabs syncKey="pkg">

   <TabItem label="npm">

   ```sh
   npm install starlight-typesense
   ```

   </TabItem>

   <TabItem label="pnpm">

   ```sh
   pnpm add starlight-typesense
   ```

   </TabItem>

   <TabItem label="Yarn">

   ```sh
   yarn add starlight-typesense
   ```

   </TabItem>

   <TabItem label="Bun">

   ```sh
   bun i starlight-typesense
   ```

   </TabItem>

   </Tabs>

2. Configure the plugin in your Starlight [configuration](https://starlight.astro.build/reference/configuration/#plugins) in the `astro.config.mjs` file.

   ```diff lang="js" ins={4,10-16}
   // astro.config.mjs
   import starlight from '@astrojs/starlight'
   import { defineConfig } from 'astro/config'
   import starlightTypesense from 'starlight-typesense'

   export default defineConfig({
     integrations: [
        starlight({
          plugins: [
          starlightTypesense({
            typesenseCollectionName: 'docs',
            typesenseServerConfig: {
              nodes: [{ url: 'http://localhost:8108' }],
              apiKey: 'xyz',
              connectionTimeoutSeconds: 5,
            },
          }),
        ],
        title: 'My Docs',
       }),
     ],
   })
   ```

3. [Start the development server](https://starlight.astro.build/getting-started/#start-the-development-server) to preview the plugin in action.

</Steps>
