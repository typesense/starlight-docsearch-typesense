`starlight-docsearch-typesense` is a Starlight plugin that integrates <a href="https://typesense.org/" target="_blank">Typesense</a>
with the <a href="https://docsearch.algolia.com/" target="_blank">DocSearch</a> interface to add lightning-fast, typo-tolerant search to your Starlight documentation site.

Itâ€™s a drop-in replacement for Algolia DocSearch, powered by an open-source search engine.

## Prerequisites

You'll need an existing Starlight project.
If you don't have one yet, follow the <a href="https://starlight.astro.build/getting-started/" target="_blank">Getting Started guide</a> to create it.

## Installation

import { Steps, Tabs, TabItem } from '@astrojs/starlight/components';

<Steps>

1. Install the plugin
   by running one of the following commands in your terminal:

   <Tabs syncKey="pkg">

   <TabItem label="npm">

   ```sh
   npm install starlight-docsearch-typesense
   ```

   </TabItem>

   <TabItem label="pnpm">

   ```sh
   pnpm add starlight-docsearch-typesense
   ```

   </TabItem>

   <TabItem label="Yarn">

   ```sh
   yarn add starlight-docsearch-typesense
   ```

   </TabItem>

   <TabItem label="Bun">

   ```sh
   bun i starlight-docsearch-typesense
   ```

   </TabItem>

   </Tabs>

2. Configure the plugin in your Starlight [configuration](https://starlight.astro.build/reference/configuration/#plugins) in the `astro.config.mjs` file.

   ```diff lang="js" ins={4,11-17}
   // astro.config.mjs
   import starlight from '@astrojs/starlight'
   import { defineConfig } from 'astro/config'
   import starlightDocSearchTypesense from 'starlight-docsearch-typesense'

   export default defineConfig({
     integrations: [
        starlight({
          title: 'My Docs',
          plugins: [
            starlightDocSearchTypesense({
              typesenseCollectionName: 'docs',
              typesenseServerConfig: {
                nodes: [{ url: 'http://localhost:8108' }],
                apiKey: 'xyz',
              },
            }),
          ],
       }),
     ],
   })
   ```

3. [Start the development server](https://starlight.astro.build/getting-started/#start-the-development-server) to preview the plugin in action.

   Once your dev server starts, search bar will appear in your site header.
   If you haven't indexed your docs yet, it will stay empty until your Typesense collection is set up.
   See the [Typesense DocSearch Scraper guide](/guides/typesense-docsearch-scraper/) on how to set up your index.

</Steps>
